# Workflow 06 — Relance après long silence (>10 jours)

> Thread avec acceptation ou engagement positif du prospect, mais silence prolongé de notre côté. Nécessite une excuse appuyée (s14b) et une approche honnête pour redémarrer proprement. Session 17/04/2026 : **2 cas** (Donavan Grondin 28j — relance active, Pierre Berdat 7 mois — skippé car hors ressort Antoine).

## Quand utiliser

- Thread où le prospect a accepté / montré un engagement positif
- Notre côté n'a pas répondu depuis **>10 jours** (s14b appuyé)
- Le prospect est toujours potentiellement intéressé (pas de signal de désengagement)
- Le dossier est dans **notre ressort** (sinon skip + note)

## Principe

- **Excuse appuyée en ouverture** (s14b) — ne pas minimiser
- **Reconnaître explicitement le retard** sans inventer d'excuse chiffrée
- **Assumer la responsabilité** ("c'est inadmissible de mon côté", "le dossier a glissé entre les mailles")
- **Proposer un redémarrage concret** — pas de "on va revoir ça"
- **Ouvrir une action simple** (demande sélection, call, décision binaire)

## Templates excuse selon le retard

| Délai | Template | Body |
|---|---|---|
| >4 jours | s14a | "Excuse-moi pour le temps de réponse !" |
| >10 jours | s14b | "Je suis vraiment désolé pour ce temps de réponse, c'est inadmissible de mon côté, c'était un peu la course !" |
| **>30 jours** | s14b + phrase bonus | "Le dossier a malheureusement glissé entre les mailles et j'en suis franchement gêné." |

**Règle** : prepend au début du message principal, ligne séparée.

## Flow step-by-step

```
1. Identifier le thread (silence >10j avec engagement passé)
2. Vérifier: est-ce dans mon ressort ? (sinon skip + note Sheet)
3. Calibrer excuse: s14a ou s14b selon délai
4. Composer: excuse + acknowledge status + proposition concrète
5. Présenter pour go ciblé (pas d'auto-envoi pour ces cas > 10j)
6. Envoyer après validation
7. Sheet: mettre à jour note K avec "relance après XX jours, s14b envoyé"
```

## Exemple 2 — Pierre Berdat (@pierreberdat_tri, 7 mois) ⚠️ hors ressort

### Contexte

- Triathlète IM World Championship 140.6 & 70.3, performances exceptionnelles (IM 9h07, marathon 2h40)
- Inbound initial 23/08/2025 — recommandé par Quentin Le Blanc et Maxime Verschoor
- Call calé → contretemps → relance 20/09/2025
- **7 mois de silence** (210 jours)

### Son dernier message (20/09/2025)

> "Hello Pierre 🙌🏼 Peut-on se programmer cela semaine prochaine?"

### Pourquoi skip

Le thread montre qu'Antoine (Impulse) avait été en contact avec **"Pierre d'Impulse"** dans cette conv — probablement **Pierre Gautier (HCS)**. Le dossier n'est pas dans le ressort d'Antoine Chabrat.

**Décision** : skip, pas de DM, pas d'ajout au Sheet Suivi_Amb (c'est un dossier Pierre).

### Leçon

Avant de drafter une relance, **toujours identifier qui est le owner du dossier** dans l'équipe Impulse/HCS :
- Antoine Chabrat (Influence Manager) : programme ambassadeur, dotations, codes affiliés
- Pierre Gautier (HCS) : sponsorings top-tier, budgets cash, décisions stratégiques
- SC (Gorgias) : tickets support client

Si un ancien message signe "Pierre d'Impulse" et que le dossier touche au sponsoring/paid → probablement Pierre Gautier → skip + flag.

## Checklist avant envoi

- [ ] Délai confirmé (>10j = s14b)
- [ ] Dossier dans mon ressort (pas Pierre / pas SC)
- [ ] Historique relu complet (scroll jusqu'au début du thread)
- [ ] Excuse calibrée (s14a / s14b / s14b + phrase bonus)
- [ ] Proposition concrète dans le message (pas "on va voir")
- [ ] Go ciblé Antoine obtenu (PAS d'auto-envoi pour ces cas)

## Red flags

| Red flag | Conséquence |
|---|---|
| Inventer une excuse chiffrée ("j'étais sur une grosse production jusqu'à hier") | Grillé si le prospect vérifie |
| Minimiser le retard ("petite attente") | Pas crédible sur 28j+ |
| Envoyer juste un "salut, tu es toujours partant ?" | Manque de reconnaissance, pro oublié |
| Prendre un dossier hors de son ressort | Doublonnage avec Pierre / autre PIC |
| Oublier le suivi Sheet après envoi | Plus de traçabilité |

## Cas limites

### Prospect a changé de situation (nouvel emploi, nouvelle marque)
→ Adapter le message en acknowledgant ce qu'on voit de son IG actuel ("vu que tu as signé avec X, j'imagine que...")

### Prospect ne répond pas à la relance
→ Attendre 2 semaines puis 2e relance neutre, puis archiver si pas de réponse

### Le prospect a un sponsor concurrent (identifié entre-temps)
→ Switch vers template s3.1 (refus poli sponsor exclusif) plutôt que s5v
