## Architecture Notes

- **No single app** — `bigblue_mcp`, `gorgias_mcp`, `shopify_mcp`, `instagram_dm_mcp` each have their own `pyproject.toml` with hatch build. Scripts at root use a bootstrapped `sys.path` to import from `common/`.
- **Google Sheets is the system of record**. Column semantics live in `common/google_sheets.py` and are documented in `docs/reference_sheet_schema.md` — update both together.
- **Four partnership models** (Affiliation, Dotation S/M/L, Paid) — see `docs/reference_contract_types.md`. Each routes to a different Sheet tab.
- **SAV draft order flow** : `docs/process_sav_unified.md` is the canonical runbook. Mandatory defaults on every replacement : `applied_discount` 100 % + `shipping_line` gratuit + `tags: Service client`.
- **Conversation archive is private** : `data/conversations/` is gitignored. Scripts anonymize before writing anything to `personality.md` or `real_response_examples.md`.
